# Sample Problems

## Text Problems

### Problem 1

```text
Solve for x: 3x - 7 = 11
```

Expected answer:

```text
x = 6
```

### Problem 2

```text
Given f(x) = 2x + 3, find f(5).
```

Expected answer:

```text
13
```

### Problem 3

```text
Which values of x satisfy x^2 - 5x + 6 = 0?
```

Expected answer:

```text
x = 2 or x = 3
```

### Problem 4

```text
A rectangle has width 4 and length 7. What is its area?
```

Expected answer:

```text
28
```
