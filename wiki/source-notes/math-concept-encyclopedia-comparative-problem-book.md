# Source Note - Math Concept Encyclopedia Comparative Problem Book

## Source Summary

이 문서는 같은 수학 개념이 한국, 일본, 중국, 미국에서 어떤 문제 감각과 표현으로 나타나는지 비교하기 위한 문제집이다.

## Notable Claims

- 같은 개념이라도 나라별로 계산 중심, 설명 중심, 그래프 중심, 모델링 중심의 차이가 드러난다.
- 개념 카드에 `비교 포인트`를 붙일 때 유용하다.
- 한국형, 일본형, 중국형, 미국형 문제를 나란히 놓아 개념의 공통 핵심과 표현 차이를 동시에 볼 수 있다.
- 이차방정식, 삼각함수, 조건부확률, 통계 해석, 회귀 같은 후속 개념 카드에도 나라별 문제 감각 차이를 붙이기 좋다.
- 일본형 비교 문제에서는 `복소수평면`과 `벡터`를 연결해 설명하게 하는 고급 표현 차이도 확인할 수 있다.
