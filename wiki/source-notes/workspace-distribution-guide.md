# Source Note - Workspace Distribution Guide

## Source Summary

이 문서는 수학 해설 에이전트 workspace를 공유할 때 공개용 자산, runnable starter scaffold, facilitator 전용 운영 메모를 어떻게 분리할지 정리한다.

## Notable Claims

- `docs/hands-on-assets/`에는 공개 가능한 handout, 템플릿, 샘플 입력만 남기는 편이 낫다.
- 실행 가능한 starter scaffold는 `starter-kits/`로 분리하는 편이 구조적으로 명확하다.
- facilitator 전용 타이밍 메모와 운영 스크립트는 shared workspace 밖에서 관리하는 편이 안전하다.
