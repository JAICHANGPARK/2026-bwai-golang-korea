# Source Note - Math Concept Encyclopedia README

## Source Summary

이 문서는 `docs/math-concept-encyclopedia/` 폴더가 나라별 수학 교육과정을 `개념 백과사전` 방식으로 읽기 위한 문서군임을 설명한다.

## Notable Claims

- 각 문서는 단원명보다 `정의`, `핵심 개념`, `대표 수식`, `중요 성질`, `연결`에 초점을 둔다.
- 나라별 개념 백과사전 외에 다국어 용어집, 수식 예제 모음, 단원별 연습문제, 비교 문제집까지 포함한다.
- 이 폴더는 커리큘럼 연구 문서와 짝을 이루며, 개념 카드형 지식 자산으로 재활용하기 좋다.
