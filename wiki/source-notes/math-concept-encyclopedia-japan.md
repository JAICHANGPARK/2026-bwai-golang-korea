# Source Note - Math Concept Encyclopedia Japan

## Source Summary

이 문서는 일본 중학교 학년별 수학과 고등학교 `수학I/A/II/B/III/C`를 `정의`, `핵심 개념`, `대표 수식`, `연결` 중심으로 다시 쓴 개념 백과사전이다.

## Notable Claims

- 중1에는 `문자식과 일원일차방정식`, `비례·반비례와 좌표`, 경험적 확률 감각이 들어간다.
- 중2에는 `연립일차방정식`, `일차함수`, `확률`이 개념 단위로 정리된다.
- 고등학교 `수학I`는 `삼각비`를 본격 도입하고, `수학A`는 경우의 수와 확률, `수학B`는 통계적 추측을 맡는다.
- `연결` 항목이 `비례 -> 일차함수 -> 이차함수`, `닮음 -> 삼각비 -> 삼각함수` 같은 계통을 분명히 보여 준다.
- 고등학교 `수학C`는 `복소수평면`과 `벡터`를 직접 연결하는 고급 좌표·대수 브리지로 읽힌다.
