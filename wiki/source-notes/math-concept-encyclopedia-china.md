# Source Note - Math Concept Encyclopedia China

## Source Summary

이 문서는 중국 `7~9학년`과 고등학교 `필수/선택필수/선택` 수학을 개념 중심으로 재구성한 백과사전이다.

## Notable Claims

- 중학교에서 `일차방정식`, `일차함수`, `예각 삼각함수`, `확률의 기초`가 비교적 빠르게 계통화된다.
- 고등학교 `확률과 통계` 축은 이후 `확률변수·정규분포·회귀`로 자연스럽게 확장된다.
- `연결` 항목이 `일차함수 -> 이차함수`, `닮음 -> 삼각비`, `확률 -> 조건부확률 -> 확률변수`처럼 다음 단계 학습을 또렷하게 드러낸다.
- 함수, 기하, 대수, 확통을 분리된 단원보다 `연결된 개념군`으로 읽기에 적합하다.
- 고등학교 확장 영역에서는 `복소수`와 `수학적 모델링·수학 탐구`가 대수·활동 축의 심화 방향을 드러낸다.
