# Source Note - Math Agent Knowledge Plan

## Source Summary

이 문서는 수학 해설 에이전트 구축 전 필요한 지식 데이터 수집 방향, 교육과정 비교, 데이터 소스 우선순위를 정리한다.

## Notable Claims

- 핵심 자산은 `문항 + 정답 + 해설 + 교육과정 + 개념 관계망`이다.
- 수능과 KICE 모의평가 데이터는 중요하지만 단독으로는 부족하다.
- 중학교 데이터, 사진형 입력 데이터, 오답 원인 데이터는 별도 보강이 필요하다.
- 2026년 기준 교육과정 적용 시점이 혼재되어 있어 버전 구분이 중요하다.
