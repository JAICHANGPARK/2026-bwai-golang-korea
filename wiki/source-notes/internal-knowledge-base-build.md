# Source Note - Internal Knowledge Base Build

## Source Summary

이 문서는 수학 해설 서비스용 내부 지식베이스를 어떤 계층과 타입으로 구축할지 정의한다.

## Notable Claims

- 내부 지식은 문제 정답을 대신 풀어주는 용도가 아니다.
- 지식은 `원천 데이터`, `정규화 데이터`, `검색용 청크`로 분리해야 한다.
- 검색 품질은 데이터 양보다 정규화와 메타데이터 일관성이 중요하다.
- 우선 지식 타입은 교육과정, 성취기준, 개념 카드, 오답 카드, 복습 카드다.
