# Geometry Strand Sample

## Summary

`기하 계통`은 `작도 -> 합동 -> 닮음 -> 피타고라스 정리 -> 삼각비 -> 도형의 방정식 -> 기하와 벡터`로 이어지는 세로 축이다. 한국 수학에서는 논증과 계산이 같이 자라기 때문에, 기하 계통은 단순 도형 목록이 아니라 `증명 언어 + 측정 언어 + 좌표 언어`가 겹치는 구조로 보는 게 좋다.

## Key Points

- 권장 진행선
  - 중1: 기본 도형과 작도
  - 중2: 평행선·다각형·합동의 논증
  - 중3: 닮음, 피타고라스 정리, [trigonometric-ratio.md](./trigonometric-ratio.md)
  - 고1: 도형의 방정식
  - 고2~고3: 기하, 벡터
- 계통 허브에서 보여줄 것
  - 직관적 도형 감각이 논증으로 바뀌는 지점
  - 닮음과 삼각비가 어떻게 계산 도구로 이어지는지
  - 좌표평면이 도형을 식으로 번역하는 지점
  - 고등 `기하` 과목과 물리·공학 응용까지의 확장
- 대표 가로 연결
  - `좌표평면 <-> 도형의 방정식`
  - `닮음 <-> 삼각비`
  - `피타고라스 정리 <-> 거리 공식`
  - `벡터 <-> 함수와 그래프`
