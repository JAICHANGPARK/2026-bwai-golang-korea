# Linear Function Sample

## Summary

`일차함수`는 중학교 함수 학습의 중심 카드로 두는 것이 좋다. 이유는 이 개념이 `비례`, `좌표평면`, `방정식`, `그래프 해석`, `기울기`, `도형의 방정식`까지 이어지는 핵심 연결점이기 때문이다.

## Key Points

- 권장 그래프 메타데이터
  - grade views: `중2`, `고1`
  - strand: `함수`
  - prerequisites: `정비례와 반비례`, `좌표평면`, `문자와 식`
  - leads to: `기울기`, `직선의 방정식`, `함수 해석`
  - misconceptions: `기울기와 비율 혼동`, `x와 y의 역할 혼동`
  - review paths: `비례에서 함수로`, `방정식과 그래프 연결`
- 왜 핵심 카드인가
  - 관계를 식, 표, 그래프로 오가며 표현하는 첫 본격 개념이다.
  - 중학교의 선형 관계가 고등학교 `함수와 그래프`, `도형의 방정식`으로 이어지는 관문이다.
  - 내신과 수능 기초 문항 모두에서 반복적으로 재등장한다.
- 추천 페이지 구성
  - `## Summary`: 일차함수의 의미와 쓰임
  - `## Key Points`: 정의, 그래프, 절편, 기울기, 해석 포인트
  - `## Connections`: [functions-strand.md](./functions-strand.md), [middle-2-hub.md](./middle-2-hub.md), 연립방정식, 직선의 방정식

- `일차함수` 카드 안에 `직선의 방정식` 연결을 직접 넣을지, 고1 전용 브리지 노드로 한 번 더 나눌지 결정이 필요하다.
