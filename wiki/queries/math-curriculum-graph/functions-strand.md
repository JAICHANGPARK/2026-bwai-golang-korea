# Functions Strand Sample

## Summary

`함수 계통`은 학년별로 끊어 읽으면 흐름이 잘 안 보이기 때문에, `비례 감각 -> 일차함수 -> 이차함수 -> 함수와 그래프 -> 극한 -> 미분`처럼 장기 계통으로 묶어야 한다. 이 축은 중학교와 고등학교를 잇는 대표적인 세로 연결선이다.

## Key Points

- 권장 진행선
  - 중1: 좌표와 비례
  - 중2: [linear-function.md](./linear-function.md)
  - 중3: [quadratic-function.md](./quadratic-function.md)
  - 고1: 함수와 그래프
  - 고2: 지수함수, 로그함수, 삼각함수
  - 고2~고3: 함수의 극한, 미분, 적분
- 계통 허브에서 보여줄 것
  - 각 개념의 선행 개념
  - 같은 개념의 표현 전환
  - 자주 생기는 오개념
  - 다음 학년에서 다시 등장하는 위치
- 대표 가로 연결
  - `일차함수 <-> 연립방정식`
  - `이차함수 <-> 이차방정식`
  - `삼각함수 <-> 삼각비`
  - `미분 <-> 함수 해석`

## Connections

- 시작점은 [middle-1-hub.md](./middle-1-hub.md)의 `좌표와 비례`다.
- 중학교 핵심 관문은 [middle-2-hub.md](./middle-2-hub.md)와 연결되는 [linear-function.md](./linear-function.md)다.
- 다음 함수 관문은 [middle-3-hub.md](./middle-3-hub.md)와 연결되는 [quadratic-function.md](./quadratic-function.md)다.
- 함수 축은 고등학교에서 `공통수학2`, `대수`, `미적분I`, `미적분II`와 강하게 연결된다.

## Open Questions

- `함수와 그래프`를 고1의 독립 허브로 둘지, 함수 계통 페이지 안의 중간 관문으로 둘지 구조 선택이 필요하다.
- `삼각함수`를 함수 축과 기하 축 어디에 더 강하게 연결할지 균형 조정이 필요하다.
