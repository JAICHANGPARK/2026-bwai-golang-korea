# Misconception Pattern Map

## Summary

오개념은 개념 카드마다 따로 적을 수도 있지만, 반복 패턴을 묶어 보면 더 빨리 보인다. 특히 `용어 혼동`, `연산 규칙 오해`, `그래프 해석 혼동`, `확률과 통계의 조건 읽기 실패`는 나라를 넘어 자주 반복된다.

## Key Points

- 수와 식
  - 등식 변형과 이항을 `부호 이동 규칙`으로만 외우다 원리를 놓치는 경우가 많다.
  - 관련 카드: [linear-equation.md](./concept-cards/linear-equation.md), [simultaneous-equations.md](./concept-cards/simultaneous-equations.md), [quadratic-equation.md](./concept-cards/quadratic-equation.md)
- 함수와 그래프
  - 기울기와 절편, 평균변화율과 순간변화율, 삼각함수의 각과 좌표를 섞어 읽는 오류가 반복된다.
  - 관련 카드: [linear-function.md](./concept-cards/linear-function.md), [derivative.md](./concept-cards/derivative.md), [trigonometric-function.md](./concept-cards/trigonometric-function.md), [trigonometric-graphs.md](./concept-cards/trigonometric-graphs.md)
- 기하와 논증
  - 합동과 닮음의 조건을 `겉모양 유사`와 혼동하거나, 증명에서 결론을 먼저 가정하는 순환 논리가 자주 나타난다.
  - 관련 카드: [congruence.md](./concept-cards/congruence.md), [similarity.md](./concept-cards/similarity.md), [geometric-proof.md](./concept-cards/geometric-proof.md)
- 확률과 통계
  - 조건부확률에서 분모를 잘못 잡거나, 상관과 인과를 혼동하거나, 표본의 대표성을 과대평가하는 경우가 많다.
  - 관련 카드: [conditional-probability.md](./concept-cards/conditional-probability.md), [correlation-and-scatter-plots.md](./concept-cards/correlation-and-scatter-plots.md), [sampling.md](./concept-cards/sampling.md), [statistical-inference.md](./concept-cards/statistical-inference.md)
