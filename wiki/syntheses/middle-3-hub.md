# Middle 3 Hub

## Summary

`중3 허브`는 중학교 수학이 고등 수학으로 넘어가기 직전 가장 압축적으로 연결되는 단계다. `실수`, `인수분해`, `이차`, `닮음과 삼각비`, `상관과 산점도`를 `고등 전이점`으로 묶어 보는 편이 좋다.

## Key Points

- 대표 개념
  - [square-roots-and-real-numbers.md](./concept-cards/square-roots-and-real-numbers.md)
  - [polynomials-and-factorization.md](./concept-cards/polynomials-and-factorization.md)
  - [quadratic-equation.md](./concept-cards/quadratic-equation.md)
  - [quadratic-function.md](./concept-cards/quadratic-function.md)
  - [similarity.md](./concept-cards/similarity.md)
  - [pythagorean-theorem.md](./concept-cards/pythagorean-theorem.md)
  - [trigonometric-ratio.md](./concept-cards/trigonometric-ratio.md)
  - [correlation-and-scatter-plots.md](./concept-cards/correlation-and-scatter-plots.md)
- 역할
  - `인수분해 -> 이차방정식 -> 이차함수`로 대수와 함수의 연결을 완성한다.
  - `닮음 -> 피타고라스 정리 -> 삼각비`로 고등 기하와 삼각함수의 입구를 연다.
  - 데이터 읽기를 고등 `확률과 통계`로 올리는 교두보다.
