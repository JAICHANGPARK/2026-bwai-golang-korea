# Example Pattern Library

## Summary

개념 카드의 예제는 `계산형`, `해석형`, `연결형`으로 나누어 두면 재사용성이 높다. `formula-examples`와 `unit-practice-book`은 이를 안정적으로 공급하는 지원 자산이다.

## Key Points

- 계산형
  - 공식 적용과 대수 조작을 확인한다.
  - 대표 카드: [linear-equation.md](./concept-cards/linear-equation.md), [quadratic-equation.md](./concept-cards/quadratic-equation.md), [complex-number-operations.md](./concept-cards/complex-number-operations.md)
- 해석형
  - 그래프, 표, 문장 상황을 읽어 개념 의미를 해석한다.
  - 대표 카드: [linear-function.md](./concept-cards/linear-function.md), [trigonometric-graphs.md](./concept-cards/trigonometric-graphs.md), [derivative-graph-interpretation.md](./concept-cards/derivative-graph-interpretation.md), [regression.md](./concept-cards/regression.md)
- 연결형
  - 한 단원에서 다음 단원으로 넘어가는 브리지를 보여 준다.
  - 대표 카드: [radians.md](./concept-cards/radians.md), [trigonometric-identities.md](./concept-cards/trigonometric-identities.md), [random-variable.md](./concept-cards/random-variable.md), [statistical-inference.md](./concept-cards/statistical-inference.md)
