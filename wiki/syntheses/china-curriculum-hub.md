# China Curriculum Hub

## Summary

중국 수학 교육과정은 `7~9학년의 빠른 계통화`와 `고교 필수/선택필수 구조`가 결합된 허브다. 중학교는 학기 단위로, 고등학교는 과정 단위로 읽는 편이 개념 재등장을 가장 잘 보여 준다.

## Key Points

- 중학교 핵심 흐름
  - [concept-cards/china-grade-7-semester-1.md](./concept-cards/china-grade-7-semester-1.md)부터 [concept-cards/china-grade-9-semester-2.md](./concept-cards/china-grade-9-semester-2.md)까지 `방정식 -> 함수 -> 삼각함수`가 촘촘하게 이어진다.
  - 데이터 축은 자료 수집, 데이터 분석, 확률의 기초 순으로 점진적으로 나타난다.
- 고등학교 핵심 흐름
  - [concept-cards/china-required-curriculum.md](./concept-cards/china-required-curriculum.md)는 함수, 기하와 대수, 확률과 통계, 모델링을 묶는다.
  - [concept-cards/china-elective-required-curriculum.md](./concept-cards/china-elective-required-curriculum.md)는 수열·도함수, 해석기하, 확률변수·정규분포·회귀를 확장한다.
  - [concept-cards/china-elective-concept-clusters.md](./concept-cards/china-elective-concept-clusters.md)는 선택 과정의 고급 개념군을 메타 수준에서 묶는다.
- 평가 메모
  - 가오카오와 직접 연결되는 구조라서 함수·기하·확통의 체계성이 특히 강하게 느껴진다.
