# Geometry Strand

## Summary

`기하 계통`은 `도형 감각 -> 논증 -> 길이와 비율 -> 좌표 번역 -> 벡터와 공간`으로 성장하는 세로 축이다. 한국 수학에서는 도형 자체보다 `이유를 설명하는 언어`, `길이를 계산하는 언어`, `좌표로 옮기는 언어`가 차례로 겹친다는 점이 핵심이다.

  - `도형의 방정식 <-> 함수 그래프`
  - `벡터 <-> 복소수평면`
