# Cross-Country Function Progression

## Summary

네 나라 모두 `비례/선형 관계 -> 이차 관계 -> 삼각/고급 함수 -> 미적분`으로 이어지지만, 언제 어느 언어로 들어가는지는 다르다. 이 차이는 개념 카드 간 간선을 단순 학년이 아니라 `전이 방식`으로 보게 해 준다.

## Key Points

- 한국
  - [proportion.md](./concept-cards/proportion.md) -> [linear-function.md](./concept-cards/linear-function.md) -> [quadratic-function.md](./concept-cards/quadratic-function.md) -> [trigonometric-function.md](./concept-cards/trigonometric-function.md) -> [limits.md](./concept-cards/limits.md) -> [differentiation.md](./concept-cards/differentiation.md)
- 일본
  - 중학교의 비례·반비례와 일차함수 연결이 매우 선명하다.
  - 고등학교에서는 [concept-cards/japan-math-1.md](./concept-cards/japan-math-1.md), [concept-cards/japan-math-2.md](./concept-cards/japan-math-2.md), [concept-cards/japan-math-3.md](./concept-cards/japan-math-3.md)가 함수 위계를 밀어 올린다.
- 중국
  - [concept-cards/china-grade-8-semester-2.md](./concept-cards/china-grade-8-semester-2.md)에서 일차함수, [concept-cards/china-grade-9-semester-1.md](./concept-cards/china-grade-9-semester-1.md)에서 이차함수, [concept-cards/china-grade-9-semester-2.md](./concept-cards/china-grade-9-semester-2.md)에서 예각 삼각함수가 이어진다.
  - 고등에서는 [concept-cards/china-required-curriculum.md](./concept-cards/china-required-curriculum.md), [concept-cards/china-elective-required-curriculum.md](./concept-cards/china-elective-required-curriculum.md)가 미분까지 연결한다.
- 미국
  - Grade 7의 비례, Grade 8의 함수와 선형모형, [concept-cards/algebra-2.md](./concept-cards/algebra-2.md), [concept-cards/precalculus.md](./concept-cards/precalculus.md), [concept-cards/ap-calculus.md](./concept-cards/ap-calculus.md)로 가는 경로형 구성이 강하다.
