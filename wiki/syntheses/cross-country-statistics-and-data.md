# Cross-Country Statistics And Data

## Summary

통계와 데이터 축은 네 나라 모두 강화되고 있지만, `자료 읽기에서 어디까지 가는가`가 다르다. 한국은 상관과 추정, 일본은 통계적 추측, 중국은 정규분포와 회귀, 미국은 middle school data와 advanced statistics track이 특히 눈에 띈다.

## Key Points

- 공통 흐름
  - [data-organization.md](./concept-cards/data-organization.md) -> [correlation-and-scatter-plots.md](./concept-cards/correlation-and-scatter-plots.md) -> [conditional-probability.md](./concept-cards/conditional-probability.md) -> [random-variable.md](./concept-cards/random-variable.md) -> [probability-distribution.md](./concept-cards/probability-distribution.md) -> [statistical-inference.md](./concept-cards/statistical-inference.md) -> [regression.md](./concept-cards/regression.md)
- 한국
  - 중학교의 산점도와 상관에서 고교 [probability-and-statistics-course.md](./concept-cards/probability-and-statistics-course.md), [practical-statistics.md](./concept-cards/practical-statistics.md)로 이어진다.
- 일본
  - 중학교의 상대도수와 확률 감각, 표본조사를 거쳐 [concept-cards/japan-math-a.md](./concept-cards/japan-math-a.md), [concept-cards/japan-math-b.md](./concept-cards/japan-math-b.md)에서 경우의 수와 통계적 추측이 강화된다.
- 중국
  - 데이터 수집과 기술에서 시작해 [concept-cards/china-elective-required-curriculum.md](./concept-cards/china-elective-required-curriculum.md)의 확률변수·정규분포·회귀까지 빠르게 확장된다.
- 미국
  - Grade 6~8에서 이미 통계와 이변량 데이터를 다루고, 고등에서 [concept-cards/statistics-and-data-analysis.md](./concept-cards/statistics-and-data-analysis.md), [concept-cards/ap-statistics.md](./concept-cards/ap-statistics.md)로 경로가 분명히 열린다.
