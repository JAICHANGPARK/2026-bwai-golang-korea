# Prompt Pack

## Summary

`docs/prompt-pack/`은 핸즈온 참가자가 순서대로 복사해 쓸 수 있는 프롬프트 모음이다. 목적은 설계 산출물을 단계적으로 만들고, 그 결과를 Build와 Antigravity에 재사용하는 것이다.

## Key Points

- 권장 순서
  - 사용자 흐름
  - 에이전트 설계
  - 시스템 프롬프트
  - 화면 설계
  - 출력 스키마
  - 실패 처리
  - 통합 구현 명세
  - Build 및 Antigravity
- 적용 범위
  - 참석자용 문서
  - 1시간 핸즈온 프로파일 기준
  - `RAG 없음`, `두 에이전트만 사용`
- 이 폴더는 운영 메모가 아니라 참가자용 artifacts에 가깝다.
