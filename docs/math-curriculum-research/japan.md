# 일본 중고등학교 수학 교육과정과 대입 준비

기준 시점은 2026년 4월이다. 이 문서는 일본의 `중학교 학습지도요령`, `고등학교 과목 구조`, `공통테스트와 대학별 시험`을 중심으로 정리한다.

## 핵심 요약

- 일본은 문부과학성(MEXT)의 `학습지도요령`에 따라 전국 공통성이 높다.
- 중학교는 `중1·중2·중3`별 목표와 내용을 비교적 분명하게 제시한다.
- 고등학교는 `학년`보다 `과목` 중심 구조가 강하다.
- 대학 입시는 `대학입학공통테스트(Common Test)`와 `대학별 개별시험`이 결합되는 이중 구조다.

## 중학교 수학

### 중1

| 영역 | 배우는 내용 |
| --- | --- |
| 수와 식 | 양수·음수, 정수 계산, 문자식, 일차식의 덧셈·뺄셈, 일원일차방정식 |
| 도형 | 기본 작도, 도형의 이동, 직선·평면의 위치 관계, 입체도형의 기초 |
| 함수 | 비례·반비례, 좌표의 의미, 표·식·그래프 |
| 데이터 활용 | 히스토그램, 상대도수, 다수 시행에서 얻는 확률의 의미 |

### 중2

| 영역 | 배우는 내용 |
| --- | --- |
| 수와 식 | 문자식의 사칙계산, 간단한 다항식, 식의 해석과 변형, 연립일차방정식 |
| 도형 | 평행선과 각, 다각형의 각, 합동, 삼각형의 합동조건, 증명의 필요성과 방법 |
| 함수 | 일차함수, 이원일차방정식과 함수의 연결, 표·식·그래프의 해석 |
| 데이터 활용 | 사분위범위와 상자수염그림, 경우의 수를 바탕으로 한 확률 |

### 중3

| 영역 | 배우는 내용 |
| --- | --- |
| 수와 식 | 제곱근, 무리수, 식의 전개와 인수분해, 이차방정식, 근의 공식 |
| 도형 | 닮음, 선분비, 원주각과 중심각, 피타고라스 정리 |
| 함수 | 이차함수 `y=ax^2`, 여러 현상과 이차함수의 연결 |
| 데이터 활용 | 표본조사 |

### 일본 중학교 수학의 특징

- `증명`이 중2에서 제도적으로 또렷하게 등장한다.
- `함수`는 중1 비례·반비례, 중2 일차함수, 중3 이차함수로 위계가 선명하다.
- `통계`는 히스토그램, 상자수염그림, 표본조사처럼 데이터 읽기와 해석 능력을 비교적 일찍부터 강조한다.

## 고등학교 수학

### 공식 과목 구조

### 공통적 성격이 강한 과목

| 과목 | 핵심 내용 |
| --- | --- |
| 수학I | 수와 식, 도형과 계량, 이차함수, 데이터 분석 |
| 수학II | 여러 식, 도형과 방정식, 지수·로그함수, 삼각함수, 미분·적분의 생각 |
| 수학III | 극한, 미분법, 적분법 |

### 선택형 과목

| 과목 | 핵심 내용 |
| --- | --- |
| 수학A | 도형의 성질, 경우의 수와 확률, 수학과 인간 활동 |
| 수학B | 수열, 통계적 추측, 수학과 사회생활 |
| 수학C | 벡터, 평면 위의 곡선과 복소수평면, 수학적 표현의 고안 |

### 대표적인 학년 배치

일본 고등학교는 학교와 진로에 따라 편차가 크지만, 전형적 배치는 보통 아래와 같다.

| 학년 | 대표 이수 과목 | 설명 |
| --- | --- | --- |
| 고1 | 수학I, 수학A | 대부분의 학생이 공통 기초를 닦는 구간 |
| 고2 | 수학II, 수학B | 일반대 진학 준비가 본격화되는 구간 |
| 고3 | 수학III, 수학C, 수험 대비 | 이공계·상위권 대학 준비생은 미적분과 벡터 비중이 커진다 |

### 일본 고등학교 수학의 특징

- `수학I`가 중학교와 고교 사이의 접속 과목 역할을 한다.
- `수학A/B/C`는 모두 선택형 구조라 학교 편제와 진로가 반영된다.
- `수학III`는 미적분 심화 과목이라 이공계 진학 준비 여부를 가르는 분기점이 된다.
- `통계적 추측`이 수학B에 자리 잡고 있어, 고교 후반에 통계 추론을 비교적 분명하게 다룬다.

## 학생들은 수학을 어떻게 학습하나

### 학교 안에서

- OECD PISA 2022 기준으로 일본 학생들은 수학 수업에서 `교사가 각 학생의 학습에 관심을 보인다`, `필요할 때 추가 도움을 준다`는 응답이 높았고, 디지털 기기 산만함 응답은 낮은 편이었다.
- 이를 학습지도요령과 함께 보면, 일본의 학교 수학은 `교사 설명`, `노트 정리`, `증명과 서술`, `정기고사 대비`가 결합된 구조로 이해할 수 있다.
- 특히 중학교에서는 비례 -> 일차함수 -> 이차함수, 합동 -> 닮음 같은 위계를 따라가며, 왜 그렇게 되는지를 설명하는 학습이 비교적 강하게 요구된다.

### 학교 밖에서

- MEXT의 `子供の学習費調査`에 따르면 공립 초등학교와 공립 중학교는 전체 학습비에서 `학교외 활동비` 비중이 60%를 넘는다.
- 같은 조사에서는 공립 초등학교 `2학년 이상`부터 보조학습비 가운데 `학습塾費(학원비)`가 가장 큰 항목으로 나타난다.
- 공립 중학교 `3학년`의 학원비는 연 `25만 엔`으로 조사돼, 학년이 올라갈수록 입시 준비형 보충학습 비중이 커지는 흐름을 보여 준다.
- OECD `Japan 2024`는 일본의 `juku`가 입시 준비와 국어·영어·수학·과학 보충학습에서 중요한 역할을 한다고 설명한다. 따라서 일본 학생들은 수학을 `학교 수업 + 숙제 + juku/가정교사`의 조합으로 공부하는 경우가 적지 않다.

## 대학 진학을 위해 준비하는 것

### 현재의 공통시험

- 현재 일본의 공통시험 공식 명칭은 `대학입학공통테스트(Common Test for University Admissions)`다.
- 대학입시센터 자료에 따르면 공통테스트는 주로 `객관식(다지선다형)` 문항으로 구성된다.
- 수학은 현재 `Mathematics (1)`과 `Mathematics (2)`로 나뉘며, 대표 과목명은 `Mathematics I, Mathematics A`와 `Mathematics II, Mathematics B, Mathematics C`다.

### 센터시험과의 관계

- 과거의 `대학입시센터시험(센터시험)`은 현행 시험이 아니다.
- 대학입시센터 연혁 자료 기준으로 `2020년 1월` 실시가 마지막 센터시험이었고, `2021년 1월`부터 공통테스트가 시행되었다.

### 일본 입시에서 수학 준비가 의미하는 것

- 공통테스트 대비: 속도, 정확도, 자료 해석, 표준화된 문제 해결
- 대학별 개별시험 대비: 서술형 풀이, 증명, 고난도 미적분·기하·수열
- 국공립대 지망생일수록 `공통테스트 + 2차 시험`의 이중 부담이 크다

### 한국과 다른 점

- 일본은 `공통시험 하나로 끝나는 구조`가 아니다.
- 상위권 대학일수록 `대학별 개별시험`의 존재감이 크다.
- 따라서 일본 학생은 `넓게 준비하는 1차 시험`과 `깊게 파는 2차 시험`을 함께 대비하는 경우가 많다.

## 일본 수학 교육의 요약

- 중학교는 학년별 위계가 아주 분명하다.
- 고등학교는 과목 선택 구조라 학교·진로에 따라 수학 경로가 갈린다.
- 입시에서는 공통테스트가 중요하지만, 대학별 시험이 여전히 강력하다.
- 네 나라 중 일본은 `국가 공통성`과 `대학별 선발의 자율성`이 동시에 강한 편이다.

## 검증 메모

- 중학교 `중1·중2·중3` 내용은 MEXT 학습지도요령 해설의 학년별 구성을 바탕으로 정리했다.
- 고등학교 `대표적인 학년 배치`는 학교 편제와 진로에 따라 달라질 수 있는 전형적 예시다.
- `센터시험 -> 대학입학공통테스트` 전환 시점은 대학입시센터 연혁 자료 기준으로 확인했다.
- `학생들은 수학을 어떻게 학습하나` 절은 OECD PISA 2022 일본 국가 노트, MEXT `子供の学習費調査`, OECD `Economic Surveys: Japan 2024`를 함께 참조해 정리했다.
