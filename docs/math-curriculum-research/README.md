# Math Curriculum Research

이 폴더에는 한국, 일본, 중국, 미국의 중고등학교 수학 교육과정과 대학 진학 준비 구조를 비교하기 위한 문서가 들어 있다.

문서마다 `공식 기준`, `대표적 학년·과목 배치`, `학생들의 실제 학습 방식`, `입시 실무에서의 의미`를 구분해 적었다. 특히 한국의 중학교 학년 배치, 중국의 중학교 단원 배열, 미국의 학년별 수학 경로는 각국 공식 문서와 대표 교재·표준을 바탕으로 재구성한 `대표 흐름`이라는 점을 함께 표시했다.

- `korea.md`: 한국 수학 교육과정과 수능 중심 대입 구조
- `japan.md`: 일본 학습지도요령과 공통테스트·대학별 시험 구조
- `china.md`: 중국 교과서 흐름, 고교 수학 표준, 가오카오 구조
- `us.md`: 미국 Common Core 기준과 대표적인 고교 수학 경로, SAT
- `comparison.md`: 네 나라 통합 비교표와 핵심 차이 요약
