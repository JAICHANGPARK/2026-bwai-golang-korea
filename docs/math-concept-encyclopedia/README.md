# Math Concept Encyclopedia

이 폴더에는 한국, 일본, 중국, 미국의 중고등학교 수학 커리큘럼을 `개념 백과사전`처럼 읽을 수 있도록 정리한 문서가 들어 있다.

각 문서는 단원이나 과목 이름만 나열하지 않고, 그 단원에서 꼭 알아야 하는 수학적 개념을 아래 형식으로 풀어 쓴다.

- `정의`: 그 단원이 다루는 대상이 무엇인지
- `핵심 개념`: 반드시 알아야 하는 용어와 사고 틀
- `대표 수식`: 자주 쓰는 식, 관계식, 함수형
- `중요 성질/공식`: 반복해서 쓰는 정리, 법칙, 계산 원리
- `연결`: 다음 단원이나 상위 과정으로 이어지는 지점

수식은 Markdown 안에서 LaTeX 문법인 `$...$`와 `$$...$$` 표기로 적었다.

## 문서 목록

- `korea.md`: 한국 중학교 대표 배치와 고등학교 과목 구조에 맞춘 개념 백과사전
- `japan.md`: 일본 중학교 학년별 구조와 `수학 I/A/II/B/III/C`에 맞춘 개념 백과사전
- `china.md`: 중국 7~9학년 대표 단원과 고교 `필수/선택필수/선택` 구조에 맞춘 개념 백과사전
- `us.md`: 미국 `Grade 6~8 + Algebra/Geometry/Precalculus/Calculus/Statistics` 경로를 기준으로 한 개념 백과사전
- `multilingual-glossary.md`: 한국어·영어·일본어·중국어를 한 줄에서 비교하는 다국어 수학 용어집
- `formula-examples.md`: 나라별 핵심 단원에 대해 블록 수식과 대표 예제를 모아 둔 교재형 부록
- `unit-practice-book.md`: 각 단원마다 3문제씩 붙인 단원별 연습문제집
- `comparative-problem-book.md`: 같은 개념을 나라별 문제 감각으로 비교해 보는 비교 문제집

## 읽는 법

- 나라별 문서는 `커리큘럼 문서`와 짝을 이루도록 만들었다.
- 같은 이름의 단원이라도 나라마다 강조점이 다를 수 있다.
- 미국 문서는 전국 단일 기준이 없으므로 `대표 경로` 기준으로 읽는 것이 적절하다.
- 한국과 중국 중학교 문서는 공식 성취기준과 대표 교과서 배열을 바탕으로 재구성한 흐름을 따른다.

## 활용 예시

- 특정 학년 단원이 `어떤 개념`을 포함하는지 빠르게 확인하기
- 다음 학년에서 갑자기 어려워지는 이유를 `개념 연결`로 파악하기
- 나라별 교과서 비교 전에 `같은 개념의 다른 배열`을 이해하기
- 문제풀이 자료를 만들 때 단원별 설명 뼈대를 잡기
